Master's Thesis: AAS-based Digital Twin for LV Distribution Grids

Developed a Digital Twin framework to solve data fragmentation in Low-Voltage grids. By implementing the Asset Administration Shell (AAS) as a Single Source of Truth (SSOT), I enabled seamless interoperability between planning (GIS/CIM) and real-time operational data.

  • Semantic Mapping: Translated CIM power system standards into the AAS metamodel.
  • Grid Observability: Validated via a 15-bus Distribution System State Estimation (DSSE) use case.
  • Impact: Achieved high-fidelity visibility at unmetered nodes with a Mean Absolute Percentage Error (MAPE) below 0.1%.
